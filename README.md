# Pavlov Internal
Showcase: [Preview](https://youtu.be/bUD7thJ8eQs)
## Features
- Silent Aimbot (Uses gun muzzle for targeting)
- Bullet TP (Teleport's gun muzzle to target player head)
- Chams (Pretty self-explanatory)
- Speed Hack
- No Recoil, Spread and Kill All
